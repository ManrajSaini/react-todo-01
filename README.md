# React Todo List

A simple Todo List web application built with React.js that allows you to manage your tasks. You can add, delete, read, and edit your todos.

## Features

- **Add Todo**: You can easily add new tasks to your list by entering them in the input field and clicking the "ADD" button.

- **Delete Todo**: Remove tasks by clicking the "Delete" button next to each task.

- **Edit Todo**: Edit the text of an existing task by clicking the "Edit" button, making changes, and then clicking "Save" or "Cancel" to update or discard the changes.

## Usage

- To add a new task, enter it in the input field and click the "ADD" button.
- To delete a task, click the "Delete" button next to it.
- To edit a task, click the "Edit" button, make changes, and click "Save" to save the changes or "Cancel" to discard them.

## Technologies Used
- **React**: A JavaScript library for building user interfaces.
- **UUID**: Used to generate unique IDs for each todo item.
